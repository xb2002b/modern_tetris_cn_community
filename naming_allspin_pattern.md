# Allspin洞的系统命名法

> 注意！这是一个新提出的理论，并未得到社区认可，也可能存在漏洞，欢迎指出问题或提出改进建议~

最后编辑于 2025.12.23

## 例1 (DT炮)

```
.##..  x
.#...  x (未变化)
##.##  5
##..#  4
#...#  3
##.##  2
##.##  1
 ^ "+0" (初始) 位置, 取第一个方块最左边的格子
```

解法表达式: **T2+0(2,3),Tr+1(1,4,5)**

|  符号   | 说明                                    |
| :-----: | --------------------------------------- |
|   T2    | T块，朝下 (四个朝向分别是 `0``r``2``l`) |
|   +0    | 第一块定义了初始位置，记为"+0"，可省略  |
|  (2,3)  | 消了整个图案的第2、3行                  |
|    ,    | 分隔符                                  |
|  Tr+1   | T块，朝右，位于初始位置右1格处          |
| (1,4,5) | 消了整个图案的第1、4、5行               |

## 例2

```
##...  x (未变化)
#....  5 (没消但有变化)
#.###  4
#..##  3
#...#  2
#.###  1
 ^
```

解法表达式: **T0(2,3\*),Ir(1,3,4,5\*)**

|    符号    | 说明                              |
| :--------: | --------------------------------- |
| T0(2,3*),  | T块，朝上，消了第2行，影响了第3行 |
|     Ir     | I块，朝右 (这里用l不影响结果)     |
| (1,3,4,5*) | 消了第1、3、4行，影响了第5行      |

## 常见问题

**Q:** "解法表达式"?

**A:** 是的，它描述的是Allspin洞的解法而不是洞的形状。反正我们可以通过表达式把洞的形状还原出来。

**Q:** 那这个表达式不唯一吧？

**A:** 是的，一个洞的表达式不唯一。就像化学命名一样，有多种方法命名同一个分子。但一旦我们有了一个表达式，就可以构造出这个洞，找出所有的解法然后根据规则在其中选择那个唯一的表达式。
